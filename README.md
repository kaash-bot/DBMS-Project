# DBMS-Project

1. Run DDL script in your MySQL shell as root with the command
        source <file_path>;
2. Similarly, run DML script.
3. Access the database you want to use in the shell by - 
        use movie_db;
4. Now, run the .jar file in your CLI with the command
        java -jar <file_path>
