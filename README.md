# DBMS-Project

* Run DDL script in your MySQL shell as root with the command
        source <file_path>;
* Similarly, run DML script.
* Access the database you want to use in the shell by - 
        use movie_db;
* Now, run the .jar file in your CLI with the command
        java -jar <file_path>
